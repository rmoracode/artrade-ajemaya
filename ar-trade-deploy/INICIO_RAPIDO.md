# ⚡ INICIO RÁPIDO

## En 3 Pasos tu App está en Internet 🚀

### 1️⃣ Instala dependencias (2 min)

```bash
npm install
```

### 2️⃣ Prueba localmente (1 min)

```bash
npm start
```

Abre: **http://localhost:3000**  
✅ Debe aparecer tu app de AR

### 3️⃣ Deploy (5 min) - Elige UNO:

#### 🟦 Vercel (Recomendado)
```bash
npm install -g vercel
vercel
```

#### 🚂 Railway (Más fácil)
1. Ve a [railway.app](https://railway.app)
2. "Create New Project"
3. "Deploy from GitHub"
4. Selecciona este repositorio
5. ¡Listo!

#### 🖥️ EasyPanel
1. Tu panel EasyPanel
2. Nuevo proyecto Docker
3. Apunta a este repositorio
4. ¡Automático!
