================================================================================
  🚀 CÓMO USAR ESTA CARPETA
================================================================================

1. COPIAR A TU MÁQUINA
   Simplemente copia la carpeta ar-trade-deploy/ a donde quieras

2. INSTALAR DEPENDENCIAS
   $ cd ar-trade-deploy
   $ npm install

3. PRUEBA LOCAL
   $ npm start
   Abre http://localhost:3000

4. SUBIR A GITHUB
   $ git init
   $ git add .
   $ git commit -m "Initial commit: AR Trade"
   $ git remote add origin https://github.com/TU-USUARIO/ar-trade.git
   $ git push -u origin main

5. DEPLOY
   Elige UNA opción:

   VERCEL (1 comando):
   $ vercel

   RAILWAY (desde railway.app):
   1. Create New Project
   2. Deploy from GitHub
   3. Selecciona tu repo
   4. ¡Listo!

   EASYPANEL (desde tu panel):
   1. Create Application
   2. Docker Repository
   3. Conecta GitHub
   4. Deploy
